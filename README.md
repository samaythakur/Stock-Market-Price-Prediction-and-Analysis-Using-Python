# Stock-Market-Price-Prediction-Analysis-Using-Python

# Project Overview
- Developed a Stock Market Price Prediction & Analysis System using Python, Machine Learning, and Deep Learning techniques for financial data analytics and forecasting.
- Collected and analyzed historical OHLCV stock market data using the yfinance API and NIFTY 50 datasets covering 2019–2024 market trends.
- Implemented comprehensive data preprocessing, feature engineering, and exploratory data analysis (EDA) using technical indicators such as MA20, MA50, RSI, MACD, and Bollinger Bands.
- Designed and trained an LSTM (Long Short-Term Memory) deep learning model using TensorFlow/Keras for next-day stock price prediction.
- Built a complete financial analytics pipeline integrating data collection, visualization, predictive modeling, portfolio analysis, and performance evaluation.

# Project Flowchart
1. Collect historical stock market OHLCV data using yfinance API and NSE datasets.
2. Perform data preprocessing:
   - Handle missing values
   - Remove anomalies
   - Normalize financial data
3. Engineer technical indicators:
   - MA20
   - MA50
   - RSI
   - MACD
   - Bollinger Bands
4. Perform Exploratory Data Analysis (EDA):
   - Correlation analysis
   - Trend analysis
   - Volatility analysis
   - Sector-wise comparison
5. Generate financial visualizations:
   - Line charts
   - Histograms
   - Heatmaps
   - Portfolio charts
6. Prepare sequential time-series data using 60-day sliding windows.
7. Train LSTM deep learning model using TensorFlow/Keras.
8. Evaluate model performance using:
   - RMSE
   - MAE
   - MAPE
   - R² Score
   - Directional Accuracy
9. Predict next-day stock closing prices.
10. Generate investment insights and portfolio optimization analysis. 

# Results & Conclusion
- Successfully developed a complete end-to-end stock market analysis and prediction system using Python and LSTM deep learning architecture.
- Achieved high prediction accuracy with:
  - RMSE: 393.86 INR
  - MAE: 326.92 INR
  - MAPE: 1.39%
  - R² Score: 0.9307
  - Directional Accuracy: 71.8%
- Demonstrated effective stock trend prediction using technical indicators such as MA20, MA50, RSI, MACD, and Bollinger Bands.
- Identified that MA20 and MA50 were the most influential predictive features for stock price forecasting.
- Successfully analyzed market behavior including volatility clustering, sector-wise performance, seasonal effects, and trading signals.
- Implemented portfolio optimization strategies showing higher returns for Realty, Energy, and Metals sectors compared to market averages.
- Demonstrated practical applications of Machine Learning and Deep Learning in quantitative finance, algorithmic trading, and investment decision support systems.
